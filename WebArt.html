Release Stress, Interactive Net Art using p5.js, 2018

Since I inspired by many net art pieces presented in the class. Such as the artist Mayer, she created a sleeping site and provide viewer dreaming experience. Also, the artwork I presented in the class name "away from vacation" and "I’m Here And There" by Jonas Lund, it's super fun when using the interactive art form to deliver the idea. Then I recognized that many many net arts usually offer interaction to the viewer and let people experience it. I am interested in this art form and ready to create a net art combine with p5js and web page development. The idea in this project is the same as my code art project which let people release the stress. This time I push it farther and create four web pages, it includes the main page which gives viewer different select options and enters the way they like to release the stress.
